## 📌 Project Title
Netflix Content Analysis: Trends in Movies and TV Shows

---

## 📖 Problem Statement
This project analyzes the Netflix Titles dataset to understand trends in movies and TV shows. The goal is to identify patterns in content distribution, release years, and ratings to gain useful insights.

---

## 📂 Dataset Information
- Source: Kaggle
- Rows: 8803
- Columns: 13

### Key Columns:
- type (Movie/TV Show)
- release_year (numeric)
- rating (categorical)
- duration

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Analysis Performed
- Data Cleaning
- Statistical Analysis
- Data Visualization (Histogram, Bar Chart, Boxplot, Heatmap)
- Machine Learning Model (Linear Regression)

---

## 📈 Key Insights
1. Most Netflix content is released after 2000.
2. Movies are more common than TV shows.
3. Older content appears as outliers.
4. No strong relationship between release year and duration.

---

## 💡 Recommendations
1. Focus on modern content production.
2. Increase TV show content for better engagement.
